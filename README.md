# Cartify - Smart Seasonal E-Commerce 🛍️

Cartify is a modern, full-stack e-commerce application built with the MERN stack. It features a unique "Smart Season" detection system tailored for the Indian climate, automatically updating its theme and product highlights based on the time of year.

## ✨ Key Features
- **🌦️ Smart Season Detection:** The app automatically detects if it's Summer, Monsoon, Winter, or Spring and updates colors, hero banners, and greetings dynamically.
- **💰 Indian Localization:** Realistic product pricing in Indian Rupees (₹) with integrated GST calculations.
- **🛒 Seamless Shopping:** Fully functional cart system with toast notifications and a responsive checkout summary.
- **📱 Modern & Responsive:** Crafted with Tailwind CSS for a premium look that works perfectly on mobile and desktop.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS, Lucide Icons
- **Backend:** Node.js, Express.js
- **Styling:** Custom Vanilla CSS + Tailwind
- **Utilities:** Axios, React Hot Toast

## 🚀 How to Run Locally
1. Clone the repository.
2. Run `npm install` in the root (this installs both client and server dependencies).
3. Run `npm run build` to prepare the frontend.
4. Run `npm start` to launch the unified server.
5. Open `http://localhost:5000` in your browser.
